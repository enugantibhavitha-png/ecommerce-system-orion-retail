# E-Commerce System — Orion Retail Management

## Overview
Designed and documented a fully functional e-commerce system 
for Orion Retail Management as part of BCIS 5120 — Information 
Systems Development at the University of North Texas. The 
project addresses the absence of an online platform for Orion 
Retail by designing a comprehensive e-commerce solution that 
improves customer experience, automates business operations, 
and enables data-driven decision-making.



---

## Problem Statement
Orion Retail Management lacked an online platform for customers 
to shop remotely. All internal operations — inventory control, 
order processing, and customer management — were paper-based, 
slow, and error-prone. This limited the company's market reach, 
customer satisfaction, and operational efficiency.

---

## Proposed Solution
A comprehensive e-commerce system encompassing:

- Interactive online storefront with product catalog, 
  search, and filtering
- Secure user account management with order history 
  and wishlists
- Shopping cart and multi-gateway checkout system 
  (credit/debit, digital wallets, multi-currency)
- Automated inventory management with real-time stock 
  tracking and low-stock alerts
- End-to-end order management from placement to delivery
- Customer support features — live chat and return/refund 
  management
- Analytics and reporting dashboard for sales, customer 
  behavior, and product performance insights

## Requirements Analysis

### Functional Requirements
- Responsive and interactive product catalog interface
- Secure user registration, authentication, and profile 
  management
- Multi-payment gateway integration (credit/debit, wallets)
- Automated inventory replenishment and stock management
- Real-time order lifecycle tracking for staff and customers
- Analytics dashboard for sales, stock, and customer trends

### Non-Functional Requirements
| Requirement | Description |
|---|---|
| Scalability | Support growing user base and product database |
| Performance | Fast page load even under high traffic |
| Security | PCI compliance, data encryption, secure transactions |
| Usability | Accessible for users with varied technical skill levels |
| Reliability | Minimal downtime, efficient error handling |
| Maintainability | Easy addition of new products, features, designs |
